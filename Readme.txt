Project: Realtor.com web automation
Scope: Automate the test case for search by location and verify the result count, property value. 

Tools used: Java, TestNG, Selenium WebDriver 3.0, Maven

Framework Details: Framework for automation is designed using Page Object Model with data driven capabilities. 

Execution:
	Prerequisites:
	1. Eclipse with Maven, TestNG plugins
	or
	2. Windows machine with JDK, Maven setup and configure to call maven from CMD with "mvn install" command.
	Note: I didn't test the same on Mac, but with JDK, Maven it should work on Mac too. 

Results:
After test execution, results will be avilable at <Project Root>/test-output
